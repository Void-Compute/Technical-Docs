# Technical-Docs

The entire documentation and history of making Omniverse Apps work on AMD.

## Documentation Structure

This repository is organized into the following directories:

- **`architecture/`** — Technical architecture and design documents
- **`reverse-engineering/`** — Reverse engineering analysis and findings
- **`compatibility/`** — Compatibility information and requirements
- **`vendor-analysis/`** — Analysis of vendor-specific implementations
- **`troubleshooting/`** — Troubleshooting guides and solutions
- **`future-work/`** — Future improvements and planned work

## Getting Started

Navigate to the relevant documentation folder based on your needs.
